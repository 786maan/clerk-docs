**Clerk Dashboard** offers various roles to manage team members' access. Different roles ensure that team members have the appropriate level of access according to their responsibilities.

### Roles and Permissions

Refer to the permissions that each role provides mapped to Dashboard resources:

| Resource | Action | Admin | Member | Developer | Support |
| - | - | - | - | - | - |
| **Users** | Read | ✓ | ✓ | ✓ | ✓ |
| **Users** | Manage | ✓ | | | |
| **Users** | Impersonate | ✓ | | | ✓ |
| **Users** | Ban | ✓ | | | ✓ |
| **Organizations** | Read | ✓ | ✓ | ✓ | ✓ |
| **Organizations** | Manage | ✓ | | | |
| **Organization settings** | Read | ✓ | ✓ | ✓ | ✓ |
| **Organization settings** | Manage | ✓ | | ✓ | |
| **Applications** | Manage | ✓ | | | |
| **Developer Resources** | Manage | ✓ | | ✓ | |
| **API Keys** | Manage | ✓ | | | |
| **Billing** | Read | ✓ | ✓ | ✓ | ✓ |

## How to Change a Team Member's Role

To change a team member's role, you must be an **Admin**. Follow these steps in the `<OrganizationProfile />` component:

1. Click on the settings icon located on the right.
1. A modal will appear.
1. Navigate to the "Members" tab.
1. You will see a list of all members and their corresponding roles.
1. Find the member whose role you want to change and click on the role dropdown.
1. Select the new role from the dropdown menu.
